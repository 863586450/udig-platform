The Layer Manager is the "controller" for a net.refractions.udig.project.ui in this MVC system.
The UI and plugins send it requests when net.refractions.udig.project.ui changes are required.

Functional Requirements
  • Acts as Controller for net.refractions.udig.project.context in the MVC sense
  • Provide FeatureSource/FeatureStore and GC access for net.refractions.udig.cores by layerRef
  • Locks Features
  • Transactions
Non-Functional Requirements
  • Hide Locking from end user - be magic
