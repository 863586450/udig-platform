GeoAPI API

The GeoAPI project publishes their latest javadocs here:

  • GeoAPI 2.0

These interfaces are available in their Java 5.0 form. We do not make complete use of the GeoAPI interfaces, only the portion for which open source implementations have been made available.

GeoTools API

The GeoTools toolkit publishes their latest javadocs here:

  • GeoTools 2.2.x

The packages you will most often encounter when developing in udig are here:

  • org.geotools.data
  • org.geotools.feature
  • org.geotools.feature.visitor
  • org.geotools.filter
  • org.geotools.filter.function
  • org.geotools.styling

When creating your own plugins you will find the following useful for problem solving:

  • CRS
  • FeatureTypes
  • Schema
  • JTS
  • DataUtilities
  • org.geotools.graph
  • org.geotools.math
  • org.geotools.validation
