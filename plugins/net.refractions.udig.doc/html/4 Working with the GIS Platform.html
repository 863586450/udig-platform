This document serves as an introduction to the Catalog API for new uDig plugin-developers. Very few plugins will make use of catalog directly, usually you will want to work with Maps, Layers and Styles. Eventually real data is involved and you will need at least some of the information provided here.

There are two other uses for this document, they are covered by the following documentation convention:

EXT: Additional information for extending the Catalog framework to handle new content
IDE: Additional information for integrating Catalog plugin with normal Eclipse IDE development
