New Object Action

Identifier:
net.refractions.udig.ui.newObjectAction

Since:
0.9

Description:
Used to register create actions that do not require user interaction. The application displays these as quick options in the File New menu and in the Toolbar New actions.

Configuration Markup:

<!ELEMENT extension (action)*>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT action EMPTY>

<!ATTLIST action

class CDATA #REQUIRED

icon  CDATA #IMPLIED

id    CDATA #REQUIRED

label CDATA #IMPLIED

>

  • class - Provided ActionDelegate may choose to implement IWorkbenchWindowPulldownDelegate to populate a submenu.
  • icon -
  • id -
  • label - Displayed in the menu or toolbar as appropraite

Examples:
The AddLayersAction and AddToNewMap action are both registered in this fashion.

API Information:
Implementation of ActionDelegate is expected
