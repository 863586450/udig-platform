Auswahl

Auswahl ist kein einfaches Konzept, da mehrere Typen von Auswahlen existieren:

  • Auswahl im Kartenlayer
  • Auswahl zur Bearbeitung
  • Auswahl im Arbeitsplatz

Auswahl im Kartenlayer

Jeder Kartenlayer besitzt einen Filter, der bestimmt, welche Features des Layers ausgewählt sind. Diese Features werden sichtbar hervorgehoben (bspw. durch gelbe Farbe).

Auswahl zur Bearbeitung

Das EditBlackboard enthält diejenigen Features, die zur Bearbeitung ausgewählt sind. Nur eines von diesen ist das "derzeit zu bearbeitende Feature" - nämlich jenes, das die Griffe an seinen Eckpunkten zeigt (und es somit u.U. erlaubt, diese Eckpunkte zu verschiebe).

Auswahl im Arbeitsplatz

Im Arbeitsplatz gibt es noch weitere Auswahlen, je nachdem, welches das gerade aktive Fenster in uDig ist. Im Projektfenster können Sie Projekte, Karten und Layer auswählen, in der Legende einen oder mehrere Kartenlayer. Die Auswahl im Karteneditor hängt vom derzeit eingestellten Werkzeug ab.

Werkzeug Auswahl
Zoom/Pan Die im Karteneditor angezeigte Karte
Information Tools Die im Karteneditor angezeigte Karte
Auswahl Der beschriebene Auswahlfilter des gewählten Kartenlayers
Bearbeitungswerkzeuge Das derzeit zum Bearbeiten ausgewählte Feature oder
den/die ausgewählten Eckpunkt(e), je nachdem, ob derzeit Eckpunkte ausgewählt sind.
