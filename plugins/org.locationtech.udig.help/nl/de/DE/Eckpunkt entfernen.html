Eckpunkt entfernen -

Das Werkzeug "Eckpunkt entfernen" kann Features (geographische Objekte) auswählen und deren Eckpunkte entfernen. Klicken Sie dazu auf ein Feature, um es auszuwählen, und klicken Sie anschließend auf die Eckpunkte, die sie löschen wollen.

Tip: Sollten sich mehrere Features überlappen, so klicken Sie mehrere Male an diese Stelle, bis das gewünschte Feature gewählt ist. (Bei jedem Klick wird ein anderes Feature gewählt.)
