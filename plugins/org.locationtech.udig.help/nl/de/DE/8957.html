Eckpunkt hinzufügen -

Das Werkzeug "Eckpunkt hinzufügen" kann Geometrien auswählen und Eckpunkte zu diesen hinzufügen. Der Arbeitsablauf ist dabei identisch wie bei der Verwendung des Geometriebearbeitungswerkzeugs:

  1. Wählen Sie das Werkzeug "Eckpunkt hinzufügen".
  2. Klicken Sie auf ein Feature (ein geometrisches Objekt)
    Tip: Sollten sich mehrere Features überlappen, so klicken Sie mehrere Male an diese Stelle, bis das gewünschte Feature gewählt ist. (Bei jedem Klick wird ein anderes Feature gewählt.)
  3. Klicken Sie auf eine Linie bzw. Umgrenzung, um an dieser Stelle einen Eckpunkt einzufügen.
  4. Doppelklicken Sie, um die Änderung zu übernehmen.
